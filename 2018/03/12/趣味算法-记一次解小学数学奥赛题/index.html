小学生的解法（归纳演绎法）

嗯，一般的小学生是不会写程序去计算的，哼，好歹哥也是北邮本科 + 研究生，不能满足于此。如果我们站在小学生的角度应该怎么解这道题呢？

思考一会儿后，得到了简单解法。嗯，既然小学生不会写代码，找规律总会吧！

让我们来看一下斐波那契对 3 取余的规律

从上表不难看出对 3 取余后的数字总是按 1,1,2,0,2,2,1,0 为周期一直循环下去。

所以到 2015 的时候 应该对应一个周期中第 2015 mod 8 个数字，即为 1

中学生的解法（初等代数推导 + 归纳演绎）

假设数列中 c 的前面两个数字为 a 和 b，那么可以证明 c mod 3 = (a mod 3 + b mod 3) mod 3，于是我们可以直接写出，从而避免了计算斐波那契数列的麻烦，取余后的数列即为1，1，2，0，2，2，1，0，1，1，…之后的解法同上。

大学生的解法（电子类专业）

作为本科北邮通信的孩纸，或者说本科电子类的同学来说，我们应该都学过一门叫《数字电路与逻辑设计》这门课，其中我们学过时序电路及 m 序列生成器。上面的递推式 c mod 3 = (a mod 3 + b mod 3) mod 3，正好对应时序表达式 c3 = (c1 mod 3 + c2 mod3) mod 3，但是具体的求解过程…以后有机会看书，给出。
