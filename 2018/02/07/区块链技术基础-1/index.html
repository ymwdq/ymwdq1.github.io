区块链技术基础 - 1

发表于 2018-02-07 | 分类于 密码学 ， 区块链

背景

现在互联网圈大家谈论的最热的几个话题除了机器学习、深度学习以外，必然非比特币、区块链莫属了。

今天讨论班，王博分享了做了一个区块链技术综述（overview），让我对区块链有了更多的了解，其实去年中旬也有了解过区块链和比特币的相关知识，甚至想自己投资一波，但是我发现区块链这一块的热点更多的是商业模式、营销模式等非技术热点，技术上要实现一个区块链或者发布新的数字货币并不难(好吧，这句话我也看得别人的 = = …)，github上也有很多开源的区块链实现，有的甚至号称几百行 Python 代码实现自己的区xziil905–p\块链，所以当时就没有深挖下去。今天王博从技术的角度对区块链、比特币进行了分析让我受益颇多。

要了解区块链，就要首先了解区块链使用的核心技术、核心思想，而密码学正式区块链技术的重要基础。这篇文章将只介绍区块链的密码学基础，不会过多的涉及区块链的具体流程、具体实施策略等。

大纲

本文将介绍如下几个区块链中使用到的密码学技术

  • 哈希技术
  • 非对称加密技术（椭圆曲线加密）
  • 数字签名技术
  • 数字证书技术
  • 零知识证明
